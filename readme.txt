This plugin adds a minimal loading spinner to your WordPress site which enhances the progressive web app experience. It works with hidden ajax requests too. But, it ignores WordPress' heartbeat. If it didn't you'd see the spinner way too often; everytime the heartbeat.js was called.

== Installation ==

From the Add Plugins page choose Upload Plugin and upload this plugin's zip file. Activate. Enjoy.
